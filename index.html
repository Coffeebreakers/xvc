Welcome to GitHub Pages.

xvc is a small video converter addon for firefox (windows). conveting for n8x0, iPod, PSP, and others.

Dependencies

  • Firefox (win)
  • mencoder
  • ffmpeg

Install

Save ZIP file on your computer, rename the extension zip to xpi, drop xpi archive to main window of firefox, confirm instalation, restart ... piece of cake! :-)
